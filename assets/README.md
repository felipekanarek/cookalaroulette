# assets/

## favicon.svg

Ícone da aba (favicon) — um **"C" laranja** encorpado, referenciado no `index.html` via
`<link rel="icon" type="image/svg+xml">`.

- **Origem**: vetor **original**, desenhado para este projeto (sem dependência de fonte).
- **Cor**: laranja da marca `#e85d29`, sobre fundo transparente.
- **Formato**: SVG, escala perfeita em qualquer tamanho (16px → grande).

> A página em si não usa imagens (a identidade é tipográfica — ver constituição v2.0.0,
> Princípio I). Este asset é apenas o favicon.
